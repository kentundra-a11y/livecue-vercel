# LiveCue Phase 4 Alerts

日本のアーティスト公式サイトからライブ・チケット情報を取得し、販売開始を忘れないためのカレンダー通知を作成するVercel用Webアプリです。

## 優先アーティスト

- くるり
- Bialystocks

## Phase 4の機能

- 公式ライブ情報の更新確認
- チケット販売開始・終了日時の抽出
- 「本日〜明日開始」「あと7日以内」の強調表示
- 販売開始1日前・1時間前の通知付き `.ics` ファイル
- 公演日時のカレンダー登録
- 販売日時を取得できない場合の手動登録
- Ticketmaster Discovery APIとの併用
- 前回確認との差分（新着）表示

## Vercel設定

Environment Variablesに `TICKETMASTER_API_KEY` を登録してください。Phase 3から更新する場合は既存設定をそのまま利用できます。

`.ics` の通知は、インポート先カレンダーの通知設定や端末設定によって動作が異なる場合があります。公式サイトの表記変更により日時を抽出できない場合は手動登録を利用してください。
